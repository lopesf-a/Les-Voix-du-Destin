# Manuel utilisateur — Les Voix du Destin

## Connexion

1. Ouvrir l’application.
2. Cliquer sur **Connexion**.
3. Utiliser un compte existant ou créer un compte.
4. Accéder au dashboard.

Compte de démonstration :

```txt
player@example.com / password123
```

## Créer un personnage

1. Aller dans **Personnages**.
2. Renseigner le nom du personnage.
3. Choisir une classe.
4. Définir les statistiques.
5. Valider la création.

## Lancer une partie

1. Aller dans **Jouer**.
2. Choisir un personnage.
3. Choisir un scénario.
4. Créer une session.
5. Envoyer une première action au maître du jeu.

## Envoyer une action

Le joueur écrit une action naturelle, par exemple :

```txt
J’observe la salle pour chercher une sortie.
```

Le backend calcule le jet de dé et l’IA raconte la conséquence.

## Consulter l’historique

1. Aller dans **Historique**.
2. Sélectionner une session sauvegardée.
3. Relire les messages précédents.

## Comprendre les résultats de dé

Le système utilise un jet `1d20`.

Le résultat prend en compte :

- la statistique utilisée ;
- le modificateur ;
- la difficulté ;
- la réussite ou l’échec ;
- les effets sur les PV ou l’expérience.
